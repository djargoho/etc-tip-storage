vsc 단축키 tip 7 - by nicolas

1. ctrl+d  
- 선택한 반복된 단어를 찾아줌 그리고 같이 고칠수도 있음 

2. alt+click 
선택한 부분을 한꺼번에 편집할 수 있음 

3. alt + 위/아래 화살표 
선택한 라인의 코드를 옮길 수 있음 

4. alt+shift+위/아래 화살표
선택한 라인의 복사 붙여넣기가 가능 

5. ctrl + / 
주석(Comment) 처리 

6. alt+shift+i 
선택한 영역의 end 부분 전체 라인마다 커서 

7. alt+shift+drag
선택한 마우스 영역의 부분의 전체 라인마다 커서
